# CryptoSafe Manager - User Guide

## Установка и запуск

1. Установите Python 3.9+.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Запустите приложение:

```bash
python run.py
```

Если используется собранная версия, откройте `CryptoSafeManager.exe` из папки `dist/CryptoSafeManager`.

## Создание хранилища

При первом запуске мастер настройки попросит выбрать путь к базе SQLite и создать мастер-пароль. Мастер-пароль не хранится в явном виде: для проверки используется соль и криптографический хеш.

## Вход в хранилище

При следующих запусках введите мастер-пароль в диалоге входа. После нескольких неверных попыток включается временная блокировка.

## Работа с записями

- **Добавить**: нажмите кнопку добавления записи и заполните поля.
- **Редактировать**: выберите запись и откройте диалог редактирования.
- **Удалить**: выберите запись и подтвердите удаление.
- **Поиск**: используйте строку поиска; поддерживаются поля, категории и теги.
- **Генератор паролей**: доступен в диалоге записи.

## Защищенный буфер обмена

Кнопки копирования логина и пароля помещают данные в системный clipboard. Буфер автоматически очищается по таймеру, при блокировке, выходе или panic mode. В статусной строке отображается состояние clipboard.

## Импорт и экспорт

Через меню или toolbar доступны диалоги импорта и экспорта. Поддерживаются native JSON, CSV и совместимые форматы Bitwarden/LastPass. Для чувствительных экспортов требуется подтверждение мастер-паролем.

## Audit log

Журнал аудита фиксирует вход, операции с vault, clipboard, импорт/экспорт, panic events и проверки целостности. Записи связаны hash chain и подписываются.

## Auto-lock и настройки безопасности

В настройках можно выбрать профиль безопасности, timeout auto-lock, чувствительность активности, поведение tray и panic mode. По умолчанию включены защищенные настройки.

## Panic mode

Режим паники можно активировать кнопкой `Режим паники` на панели инструментов, пунктом меню `Файл -> Режим паники`, через tray, жестом встряхивания окна или горячей клавишей `Ctrl+Alt+P`. Для жеста возьмите окно за заголовок и быстро переместите его влево-вправо несколько раз. Panic mode блокирует vault, очищает clipboard и защищенную память, закрывает дочерние окна и может выполнить скрытные действия.

## Типовые проблемы

- Если clipboard не работает на Linux, установите `wl-clipboard`, `xclip` или `xsel`.
- Если QR scanning недоступен, проверьте установку `pyzbar` и `opencv-python`.
- Если приложение не запускается из exe, запустите его из исходников и проверьте зависимости.
